# StudentApplication

## ■ 概要

本プロジェクトはAndroidアプリ開発学習用のハンズオン教材です。  
Activity、ListView、Intent、データ保存などの基本技術を学習します。

---

## ■ 学習目標

- Activityの基本構造を理解する
- 画面遷移（Intent）の仕組みを理解する
- ListViewを使った一覧表示を実装できる
- データ保存（Preferences / File / Room）を理解する
- 非同期処理（Coroutine）の基本を理解する

---

## ■ 開発環境

- Android Studio：Koala 2024.1.1 以上
- Kotlin：1.10.x
- compileSdk：36
- minSdk：27
- 実行環境：エミュレーター または 実機

---

## ■ 起動方法

1. Android Studioで本プロジェクトを開く  
2. Gradle Syncを実行する  
3. エミュレーターを起動する  
4. Runボタンでアプリを実行する  

---

## ■ プロジェクト構成

### MainActivity
メニュー画面（各ハンズオンへの入口）

### LayoutActivity
画面レイアウトの学習

### ListActivity
一覧表示の学習

### ImplicitIntentActivity
暗黙的Intentの学習

### CameraActivity
カメラ機能の利用

### PermissionActivity
パーミッション管理の学習

### StorageActivity
ファイル操作の学習

### PreferencesActivity
設定ファイルの学習

### UserSelectActivity
Roomデータベースの学習

### CoroutineActivity
非同期処理の学習
